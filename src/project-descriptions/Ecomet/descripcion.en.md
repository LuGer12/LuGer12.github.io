# Ecomet: Excursion Booking Platform for El Ávila

Ecomet is a website to schedule excursions to Cerro El Ávila. It lets users book spots, view route details, dates and meeting points, and manage confirmations and notifications.

## Technologies

- JavaScript: 64.3%
- CSS: 35.5%
- HTML: 0.2%

Built with React.js and Firebase (authentication, real-time database, and hosting).
