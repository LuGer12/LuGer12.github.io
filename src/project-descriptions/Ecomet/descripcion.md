# Ecomet: Plataforma para Agendar Excursiones al Ávila

Ecomet es un sitio web para agendar excursiones al Cerro El Ávila. Permite a los usuarios reservar plazas, consultar detalles de las rutas, fechas y puntos de encuentro, y gestionar confirmaciones y notificaciones.

## Tecnologías

- JavaScript: 64.3%
- CSS: 35.5%
- HTML: 0.2%

Frameworks y servicios: React.js y Firebase (autenticación, base de datos en tiempo real y hosting).
