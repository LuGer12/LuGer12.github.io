# SimonDice: Juego de Memoria Interactivo
SimonDice es una versión moderna del clásico juego de memoria: reproduce secuencias de luces y sonidos para desafiar la agilidad mental. Pensado como una pequeña experiencia lúdica y educativa, combina animaciones sencillas, retroalimentación sonora y controles táctiles para ofrecer diversión inmediata en navegador y dispositivos móviles.

## Stack
JavaScript (59.3%), CSS (25.6%), HTML (15.1%)
