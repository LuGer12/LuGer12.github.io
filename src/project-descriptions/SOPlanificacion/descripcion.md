# SOPlanificacion: Sistema Operativo de Planificación
SOPlanificacion es un sistema operativo educativo donde se implementan y prueban distintos algoritmos de planificación de procesos (por ejemplo: FCFS, SJF, Round-Robin, prioridades). Permite simular la ejecución de procesos, visualizar colas, y medir métricas como tiempo de espera y turnaround.

## Tecnologías

- Java (implementación principal)
